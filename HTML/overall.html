Required

used to make a form field mandatory. The user cannot submit the form without filling that field

label

tag is used to provide a description for form input fields. It improves readability, accessibility, and user interaction by connecting labels with input elements

Text area

tag is used to get large text input from the user. It supports multiple lines, so users can type messages, comments, feedback, or descriptions easily

select

tag is used to create a dropdown list in HTML. It allows users to choose options from a list. The option tag is used inside the select tag to add choices.

option

Tag is used inside the select tag to define the choices in a dropdown list

option group

tag is used to group related options inside a dropdown list

field set

it is used to group related form elements together inside a form. It creates a box around the fields and makes the form look more organized and easy to understand. It is commonly used in registration forms, contact forms, and login forms to separate different sections of information.

legend

it is used inside the fieldset tag to provide a title or caption for the grouped section. It helps users understand what type of information belongs to that group of fields. Together, fieldset and legend improve the structure, readability, and appearance of forms.

table

it is used to display data in rows and columns. It helps organize information in a structured forma

which contain tabel head and tabel body

row span

rowspan is used to merge cells vertically across multiple rows. It allows one cell to occupy more than one row.

colspan

colspan is used to merge cells horizontally across multiple columns. It allows one cell to occupy more than one column.

i frame

it is used to display or embed another webpage, video, map, or external content inside the current webpage.

meta tags

it is used to provide information about a webpage to browsers and search engines. They are placed inside the head section of an HTML document and are not directly visible on the webpage.

meta charset

it is used to define the character encoding of a webpage. It tells the browser how to display text and special characters correctly.

meta description

it is used to provide a short summary of the webpage content. Search engines often display this description in search results, which helps improve SEO and user understanding of the page.

meta keywords

it is used to specify important keywords related to the webpage content. It was mainly used for search engine optimization to help search engines identify the page topic.

meta author

it is used to define the name of the author or creator of the webpage. It provides information about who developed or wrote the content.

og

OG stands for Open Graph. Open Graph meta tags are used to control how a webpage appears when it is shared on social media platforms like Facebook, LinkedIn, WhatsApp, and Twitter

og:title

it is used to define the title of the webpage that appears when the link is shared on social media platforms.

og:description

it is used to provide a short description or summary of the webpage content in the shared preview.

og:image

it is used to specify the image that should be displayed when the webpage link is shared.

og:url

it is used to define the official URL or link of the webpage for social media sharing.

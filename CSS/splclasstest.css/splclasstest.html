1.What is CSS and why is it used?
  1. CSS (Cascading Style Sheets) is used to style and design web pages.
  2. It controls colors, fonts, spacing, layouts, and responsiveness.
  3. It separates presentation from content.

2.What are the three types of CSS?
  1. Inline CSS
  2. Internal CSS
  3. External CSS

3.What is the difference between Inline CSS, Internal CSS, and External CSS?
  1. Inline CSS: Applied directly to an element.
  2. Internal CSS: Written inside a style section within a single page.
  3. External CSS: Written in a separate CSS file and linked to pages.
  4. External CSS is reusable across multiple pages.

4.Which CSS type has highest priority?
  1. Inline CSS has the highest priority.
  2. Then Internal CSS.
  3. Then External CSS.

5.Which CSS method is mostly used in real-time projects?
  1. External CSS.
  2. It improves maintainability, reusability, and code organization

6.What are CSS selectors?
  1. CSS selectors are used to target HTML elements for styling.
  2. They help apply styles to specific elements or groups of elements

7.What is the difference between class selector and ID selector?
  1. Class selector can be used on multiple elements.
  2. ID selector should be unique for a single element.
  3. ID selectors have higher specificity than class selectors.

8.What is the purpose of universal selector in CSS?
  1. The universal selector selects all elements on a page.
  2. It is commonly used for applying global styles or resetting default browser styles.

9.What is cascading in CSS?
  1. Cascading is the process CSS uses to determine which style rule should be applied when multiple rules target the same element.
  2. It considers priority, specificity, and source order.

10.What is specificity in CSS?
  1. Specificity is the ranking system used to determine which CSS rule takes precedence.
  2. More specific selectors override less specific selectors.

11.What is the difference between block and inline elements?
  1. Block elements occupy the full available width and start on a new line.
  2. Inline elements occupy only the required width and remain within the same line.

12.What is Flexbox and why is it used?
  1. Flexbox is a layout model used to align and distribute items efficiently.
  2. It simplifies responsive layouts and element alignment.

13.What is the difference between justify-content and align-items?
  1. justify-content aligns items along the main axis.
  2. align-items aligns items along the cross axis.

14.What is the difference between px, em, rem, and percentage units?
  1. px: Fixed unit.
  2. em: Relative to the parent element's font size.
  3. rem: Relative to the root element's font size.
  4. %: Relative to the parent element's size.

15.What is the difference between RGB, RGBA, and HEX color values in CSS?
  1. RGB: Uses red, green, and blue color values.
  2. RGBA: Uses RGB values plus transparency (alpha).
  3. HEX: Uses hexadecimal notation to represent colors.

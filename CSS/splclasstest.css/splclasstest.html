1. What is CSS and why is it used?
CSS (Cascading Style Sheets) is a styling language used to control the appearance and layout of web pages. It works together with HTML to make websites visually
Uses of CSS:
  1. Adding colors, fonts, and backgrounds
  2. Controlling spacing and alignment
  3. Creating responsive designs
  4. Improving user experience
  5. Separating design from content
2. What are the three types of CSS?
The three types of CSS are:
Inline CSS
Internal CSS
External CSS

3. What is the difference between Inline CSS, Internal CSS, and External CSS?
Inline CSS Internal CSS External CSS
Applied directly to an HTML element Written inside a style section within the HTML page Written in a separate CSS file
Affects only one element
Affects a single web page
Can affect multiple web pages
Difficult to maintain
Suitable for small pages
Best for large projects

4. Which CSS type has the highest priority?
Inline CSS has the highest priority among the three CSS types because it is applied directly
Priority order:
Inline CSS
Internal CSS
External CSS

5. Which CSS method is mostly used in real-time projects?
External CSS is mostly used in real-world projects because:
It keeps code organized.
It can be reused across multiple pages.
It is easier to maintain and update.
It improves website performance.

6. What are CSS selectors?
CSS selectors are patterns used to select HTML elements that need styling.
Common selectors include:
Element selector
Class selector
ID selector
Universal selector
Group selector

7. What is the difference between a class selector and an ID selector?

8. What is the purpose of the universal selector in CSS?
The universal selector selects all elements on a web page.
Purpose:
Applying common styles to every element
Resetting default browser margins and padding
Creating consistent styling across the page

9. What is cascading in CSS?
Cascading is the process CSS uses to determine which style rule should be applied when multiple rules target the same element.
The decision is based on:
Priority
Specificity
Order of appearance
The most appropriate rule is applied to the element.

10. What is specificity in CSS?
Specificity is a ranking system used to determine which CSS rule has higher importance when multiple rules target the same element.
General order of specificity:
Inline styles
ID selectors
Class selectors
Element selectors
Higher specificity rules override lower specificity rules.

12. What is Flexbox and why is it used?
Flexbox (Flexible Box Layout) is a CSS layout model designed to arrange elements efficiently within a container.
Why it is used:
Creates responsive layouts
Aligns items easily
Distributes space between elements
Simplifies horizontal and vertical centering
Reduces the need for complex layout techniques

justify-content align-items

Aligns items along the cross axis

Usually controls vertical alignment in a row layout
Manages spacing between items
Controls item positioning across the container

14. What is the difference between px, em, rem, and percentage units?
Unit Description
px Fixed-size unit
em Relative to the parent element's font size
rem Relative to the root element's font size
% Relative to the size of the parent element
Key Point:
px is fixed, while em, rem, and percentage are relative units that help create responsive designs.
